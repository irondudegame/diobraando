Ban Appeals

If you have been banned from the server and would like to submit an appeal:

Method 1: Discord

If you have access to the Discord, you may submit an appeal on #ban-appeal. Remember to follow all the instructions and rules or your appeal will be rejected immediately. Don't appeal more than twice.

Method 2: Forms

This method is currently in development.



Mute Appeals

If you have been temporarily muted on the server, there is no need to submit an appeal. However, if you will still like to appeal:

Method 1: Discord

If you have access to the Discord,you may submit an appeal on #mute-appeal.
